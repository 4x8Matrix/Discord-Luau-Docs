## Properties

### Style

 ```lua
 ButtonBuilder.Style :: [table](https://www.lua.org/pil/2.5.html)
```


An enumeration of button styles.

- Blurple: 1
- Grey: 2
- Green: 3
- Red: 4
- Link: 5
